# Ипотечный калькулятор

Приложение представляет собой простой калькулятор ипотеки, рассчитывающий процентную ставку, ежемесячный платёж и сумму кредита

### Install the dependencies:
```
$ npm install
```
### Start an app:
```
$ npm run dev
```
### Make a build
```
$ npm run build
```
